Animation¶

Animation makes CAE application, a complete one. VCollab provides very user friendly  and smooth animation.

User can control,

  • Speed of animation

  • Direction of animation

  • Frames per second using Animation settings

Show previous frame in the animation

Pauses the animation.

Play Animation in forward direction

Show next frame in the animation

Allows user to control frame rate (speed) of the animation.

Displays Current frame against total number of frames.

Link to CAE Animation Settings panel

Closes the panel.

Note

  • This animation is based on and for selected CAE model if more than one model exists.
