Contour Lines¶

This option shows or hides the contour lines.

  • Click ‘Contour Lines’ icon to enable.

  • Notice that contour zones are created and displayed.

  • Line color is white by default.

  • Open ‘CAE Settings’ icon to change the contour line color.

  • Click ‘Options’ tab.

  • Click the color window straight against ‘User Color’ option under Contour Lines.

  • It opens color palette window.

  • Select a color and click OK to apply.

  • Unselect ‘User Color’ option to apply palette fringe color to contour lines.
