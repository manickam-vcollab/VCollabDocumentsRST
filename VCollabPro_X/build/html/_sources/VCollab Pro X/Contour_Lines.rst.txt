Contour Lines
==============

This option shows or hides the contour lines.                         
                                                                         
   -  Click 'Contour Lines' icon |image1| to enable.                     
   -  Notice that contour zones are created and displayed.

      |image2|

   -  Line color is white by default.                                    
   -  Open 'CAE Settings' icon |image3| to change the contour line       
      color.                                                             
   -  Click 'Options' tab.

      |image4|

   -  Click the color window straight against 'User Color' option under  
      Contour Lines.                                                     
   -  It opens color palette window.                                     
   -  Select a color and click OK to apply.

      |image5|

   -  Unselect 'User Color' option to apply palette fringe color to      
      contour lines.

      |image6|
