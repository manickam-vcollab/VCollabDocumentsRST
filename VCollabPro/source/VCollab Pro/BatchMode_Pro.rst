Batch Mode
==========

VCollab Pro
-----------

**XML Based Inputs**

    VCollabPro.exe can be run as batch mode, by passing xml based inputs
    as options.

    E.g. C:\:> ...\VCollabPro.exe -b "input.xml" "arg1" "arg2"
    ...... "argN"

    VCollab provides batch mode execution for the following options.
