VCollabPro 
=============

VCollabPro is a CAX authoring tool. Using VCollabPro , the CAX can be edited/enhanced. CAE reports can be authored using VCollabPro X by creating ViewPoints.Multiple CAX files can be merged into a single CAX file for multi-disciplinary visualization.AX file from CATIA, CAX file from ABAQUS and CAX file from Fluent can be merged into one single CAX file. 3D Slides ( Viewpoints) and 3D walkthrus can be created and saved into CAX, for improved 3D presentations, documentation and easier sharing.
