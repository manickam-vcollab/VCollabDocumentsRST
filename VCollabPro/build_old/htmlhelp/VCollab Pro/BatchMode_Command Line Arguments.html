Loading CAX files

The following command line arguments are available for loading CAX files in VCollab Pro.

  1. To load a CAX File,

    C:\>...VCollabPro.exe [cax filename]

    Example:

    C:\>...VCollabPro.exe "G:DataMyFile.cax"

  2. To load two CAX files,

    C:\>...VCollabPro.exe [cax filename1] [cax filename2]

    Example:

    C:\>...VCollabPro.exe "G:DataMyFile1.cax" "G:DataMyFile2.cax"

Note

Users can load a maximum of two CAX files using command line arguments.
