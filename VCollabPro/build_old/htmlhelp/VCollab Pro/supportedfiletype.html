Supported File Types in VCollab Pro

The file types supported by VCollab Pro are listed below.

Input Files

  • VCollab CAX files (.cax)

Output Files

3D file formats:

  • VCollab CAX files (*.cax)
  • VRML 2.0 files (*.wrl)
  • STL files (*.stl)
  • JT files (*.jt)
  • 3DPDF (*.pdf)
  • FBX Files (*.fbx)
  • WCAX Files (*.wcax)
  • 3D HTML Files (*.html)

Image file formats:

  • BMP Image (*.bmp)
  • JPEG Image (*.jpg)
  • TIFF Image (*.tiff)
  • PNG Image (*.png)

Motion file formats:

  • Animated GIF (*.gif)
  • AVI Files (*.avi)
  • MP4 Files (*.mp4)
  • WMV Files (*.wmv)
  • FLV Files (*.flv)
  • SWF Files (*.swf)

Other formats:

  • Viewpoint Files (*.vpt)
  • Hotspot files (*.hotspot)
  • Microsoft Powerpoint Presentation Files (*.ppt)
  • Comma Separated Values Files (*.csv)
