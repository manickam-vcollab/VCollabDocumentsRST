Applicability of Presenter in Browsers and Microsoft Office
============================================================

Case 1:

    Operating System : Windows 7 and above, 32-bit

    MS Office : 32-bit

    VCollab Presenter : 32-bit

|     Application                |     32/64 bit   | Supported?   |

|     Internet Explorer          |     32          | Yes          |

|                                |     64          | N/A          |

|     Firefox                    |     32          | Yes          |
