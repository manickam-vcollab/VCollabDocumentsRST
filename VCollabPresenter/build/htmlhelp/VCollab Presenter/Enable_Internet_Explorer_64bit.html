Enabling 64 Bit Internet Explorer

Steps for enabling 64 bit Internet Explorer

  1. Go to Control Panel | User Accounts.
  1. Click Change User Account Control Settings.
  1. Keep Default settings.
  2. Go to Control Panel | Internet Options which opens the Internet Properties panel.
  1. Select Advanced tab.
  1. Select Allow active content to run in files on My Computer* under Security.
  1. Select Enable 64-bit tab processes for Enhanced Protected Mode*.

  1. Select Enable Enhanced Protected Mode*.

  1. Click OK.
