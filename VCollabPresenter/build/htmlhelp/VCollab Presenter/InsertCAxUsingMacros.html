Embedding CAX file using Macros in MS Office

Macros in MS office are used to automate tasks which are repetitive thereby speeding up the process.

The CAX files generated by VCollab products can be inserted in MS office files using macros.

Using Macros in MS Word 2016

  • Go to Tools->Macro->Macros Menu

  • Enter a name for the new macro and click Create.

  • A Visual basic editor window opens. Copy and paste the following lines for the InsertCAX function.

  • Provide correct file path for the input CAX file in .FilePath line instead of C:\MyCaxFilePath.cax

    Sub InsertCAX()

    ‘

    ‘ InsertCAX Macro

    ‘ Macro recorded 1/8/2010 by vct user

    ‘ActiveDocument.ToggleFormsDesign

    Set vcollab = Selection.InlineShapes.AddOLEControl(ClassType:=”VCOLLAB.VCollabCtrl.1”)

    With vcollab.OLEFormat.Object

    .FilePath = “C:\MyCaxFilePath.cax”

    End With

    End Sub

  • Close VC Editor Window.

  • Click Macros in the Developer tab.

  • Select and Run the macro created.

VCollab Presenter control is thus inserted.
