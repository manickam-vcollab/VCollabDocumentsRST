Origin
=======

This command display scene origin with axis triad. User can scale the 
triad up to model bounding sphere radius.                             
                                                                          
**How to display World origin axis Triad?**                
                                                                          
    -  Click Origin icon |image1| in Edit toolbar.                        
    -  Select Settings option in the dropdown menu, to scale up and down  
       the axis Triad.
